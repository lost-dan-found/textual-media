# Glancer

**Glancer** is a sleek terminal-based dashboard that brings live information—like time, weather, and custom messages—directly to your CLI. Designed for coders and power users who love keeping tabs on their environment without leaving the terminal.

---

## Features

- Real-time **clock** for your local timezone
- Live **weather updates** (supports Open-Meteo)
- Fully **terminal-friendly**, works with Nerd Fonts for cool icons
- **Customizable** layout and widgets via CSS-like styling in Textual
